# NewsApp
A simple demo imitating ZhiHu
网络请求使用Okhttp框架，图片加载使用Glide框架。

# 首页
模仿知乎日报的界面
JSON解析使用了GSON和JSONObject
为RecyclerView添加了header轮播图
轮播图使用ViewPager实现，自动轮播通过Handler延时发送Message实现
加载到最底部第六个Item时自动加载更多
SwipeRefreshLayout实现下拉刷新
